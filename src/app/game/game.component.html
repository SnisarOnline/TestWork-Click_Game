### Для успешного прохождения нашего теста вы должны соответствовать следующим критериям:
- игра ДОЛЖНА работать без ошибок
- предустановки сложности игры, которые вы можете получить с помощью этой конечной точки this-domain / game-settings. Это должен быть запрос к серверу, а НЕ просто копирование и вставка данных. > Эти данные представляют собой массив объектов с двумя полями:
- field - содержат размер игрового поля. Должен быть квадратной формы
- delay - время в миллисекундах
- данные для таблицы лидеров вы можете получить из этого домена / победителей. Также это должна быть просьба.
- когда кто-то выигрывает, вы должны отправить запрос на сервер с JSON, который содержит два поля победителя и дату
- пользователь устанавливает сложность игры и название
- Нажмите кнопку PLAY
- через заданный промежуток времени (в поле задержки) случайный квадрат на поле подсвечивается синим цветом
- если пользователю удалось за это время щелкнуть по квадрату - он становится зеленым, игрок получает очко, а поле меняет цвет на зеленый
- если нет, поле становится красным и точка переходит в компьютер
- когда игрок или компьютер закрашивает> 50% всех возможных квадратов в свой цвет - он становится победителем
- между кнопками управления и игровым полем появляется надпись, которую выиграл игрок (имя, которое он ввел) / компьютер
- кнопка PLAY изменяет заголовок на PLAY AGAIN
- результат игры необходимо отправить на сервер на этой конечной точке this-domain / winners в формате JSON с двумя полями, победителем и датой обеих строк.
- результаты в таблице должны обновляться автоматически
